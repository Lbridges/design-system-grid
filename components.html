Custom Classes for This Hero Background Image

  • home-bg-1 Attach this class to the section that will span the width of the screen
  • hero-content Use this class to target children of the container div in the hero section
  • col-lg-4 | col-lg-8 This is the column structure for the content inside of this exact hero content container
  • The col-lg-4 is the div that holds the text that helps define this section. The -lg-4 indicates that this content will stack under 992px
  • The col-lg-8 div is this whole ul. It is the direct child of div.col-lg-8.
  • We could target this li with .hero-content li.
